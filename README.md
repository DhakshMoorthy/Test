# Wrightfully Fit — Premium Agency Website

Luxury fitness brand website for [Wrightfully Fit](https://www.wrightfullyfit.com/).

## Stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS v4 · GSAP · Framer Motion · Lenis · React Icons

## Start

```bash
npm install
npm run dev:clean
```

Open **http://localhost:3000**

## Production

```bash
npm run build:clean
npm start
```

## Features

- Cinematic full-screen hero with parallax & Ken Burns
- Apple-style horizontal pillar showcase (Gym / Retreat / Shop)
- 11 handcrafted sections with GSAP scroll animations
- Lenis smooth scrolling, magnetic CTAs, masonry gallery lightbox
- Official Wrightfully Fit logo & photography (unchanged branding)
- Instagram brand-color edge tab, mobile sticky CTA
